Для запуска проекта необходимо
1- Скачать проект.
2- Открыть проект в VisualStudio.
3- Сделать сборку решения.
4- Запустить OrderService.ApiHost.
UI - http://localhost:58978/
Back - http://localhost:58977/swagger

Второй вариант запуска
1- Зайти в папку Build.
2- Запустить OrderService.ApiHost.exe в папке back
3- Выполнить команду "npm run dev" в папке dist
